#ERP Corporation - All-in-one Enterprise Management System

ERP Corporation là nền tảng quản trị doanh nghiệp toàn diện, giúp số hóa quy trình vận hành, nâng cao hiệu suất làm việc và tối ưu hóa nguồn lực con người trên một giao diện duy nhất.

#🚀 Các tính năng chính

#Hệ thống được xây dựng với 4 phân hệ cốt lõi:

- Task Management (Quản lý công việc): Theo dõi tiến độ dự án, phân công nhiệm vụ, quản lý deadline và quy trình làm việc (Kanban/Gantt chart).
  
- Internal Messaging (Nhắn tin nội bộ): Kênh giao tiếp thời gian thực, hỗ trợ chat nhóm, chia sẻ tài liệu và thông báo tức thì.
  
- Human Resource Management - HRM (Quản lý nhân sự): Quản lý hồ sơ nhân viên, chấm công, tính lương và đánh giá hiệu suất (KPI).
  
- Learning Management System - LMS (Quản lý đào tạo): Hệ thống lưu trữ bài giảng, tổ chức khóa học nội bộ và theo dõi lộ trình phát triển kỹ năng của nhân viên.
  
  
#🛠 Công nghệ sử dụng (Frontend)Dự án hiện đang ở phiên bản v2 (Frontend) với các công nghệ chính:

-Framework: React.js / Next.js (Tùy theo cấu trúc dự án của bạn).

-State Management: Redux Toolkit / React Query.

-UI Library: Tailwind CSS / Ant Design / Material UI.

-Communication: WebSockets (cho tính năng nhắn tin).


#📦 Cài đặtbash
 Clone dự án : git clone https://github.com

 Di chuyển vào thư mục dự án : cd erp-corporation-fe-v2

 Cài đặt các gói phụ thuộc : npm install 

 Khởi chạy dự án ở môi trường development : npm run dev 
 
#🏗 Cấu trúc thư mục (Dự kiến)
-/src/components: Các thành phần giao diện dùng chung.

-/src/modules/task: Xử lý logic và UI cho quản lý công việc.

-/src/modules/chat: Hệ thống nhắn tin.

-/src/modules/hrm: Quản lý nhân sự.

-/src/modules/lms: Quản lý học tập.
